# website
Building the Website using Bitnami and Mariadb with Jenkins in Docker


# USE Docker Compose to Launch wordpress and mariadb

Step 1: Create a docker-compose.yml file

Step 2: Create a container and Pull Mariadb image from dockerhub

Step 3: Create the env variable for the mariadb

Step 4: Create and Link volume for the mariadb

Step 5: Create a container and Pull bitnami wordpress image from dockerhub

Step 6: Create the env variable for the wordpress

Step 7: Create and Link the volume for the bitnami wordpress
